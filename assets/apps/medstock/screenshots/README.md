# MedStock App Screenshots

This folder contains the app screenshots for the MedStock gallery.

## Expected File Names

Place your screenshots here with these filenames:

1. `1-splash.png` - App splash/loading screen
2. `2-home.png` - Main home screen with features
3. `3-settings-theme.png` - Settings theme selection
4. `4-settings-language.png` - Settings language selection
5. `5-first-aid.png` - First aid kit screen
6. `6-todays-doses.png` - Today's doses screen
7. `7-medicine-details.png` - Medicine details screen
8. `8-my-medicines.png` - My medicines list screen

Once you place the PNG files here, they will automatically appear in:
- `/medstock.html` - Full gallery view
- `/apps.html` - Grid view in the featured app section

If no screenshots are found, the feature graphic will display as fallback.
